## Vue Example Files
User asked for a Vue example file and a feature breakdown table. Both are Vue Composition API focused.

### Example Vue Component
```vue
<script setup>
import { ref, computed, onMounted } from 'vue'

const user = ref({
  name: 'Lisa Park',
  bio: 'Frontend developer & coffee enthusiast',
  followers: 1204
})

const isFollowing = ref(false)

const followerCount = computed(() => {
  return isFollowing.value ? user.value.followers + 1 : user.value.followers
})

function toggleFollow() {
  isFollowing.value = !isFollowing.value
}

onMounted(() => {
  console.log('UserCard mounted')
})
</script>

<template>
  <div class="user-card">
    <h2>{{ user.name }}</h2>
    <p>{{ user.bio }}</p>
    <p>Followers: {{ followerCount }}</p>
    <button @click="toggleFollow">
      {{ isFollowing ? 'Unfollow' : 'Follow' }}
    </button>
  </div>
</template>

<style scoped>
.user-card {
  padding: 1rem;
  border: 1px solid #ddd;
  border-radius: 8px;
  max-width: 300px;
}

button {
  margin-top: 0.5rem;
  padding: 0.4rem 1rem;
  cursor: pointer;
}
</style>
```

### "What's going on here" Table

| Feature | What's going on |
|---|---|
| `<script setup>` | Syntactic sugar for Composition API — cleaner, less boilerplate |
| `ref()` | Wraps a value in a reactive container — access/mutate via `.value` |
| `computed()` | Derived state — recalculates automatically when dependencies change |
| `onMounted()` | Lifecycle hook — runs after the component is added to the DOM |
| `{{ }}` | Template interpolation — renders reactive data into HTML |
| `@click` | Shorthand for `v-on:click` — binds a DOM event to a method |
| `scoped` | Scopes CSS to this component only — no bleed into other components |
